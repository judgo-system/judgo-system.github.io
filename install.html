How to install Judgo System?

1- Install Application with Docker

2- Install Application without Docker

## Install Application without Docker Step 0- After clone project, create an activate a new virtual environment and run: ``` pip install -r requirements/base.txt ``` Step 1- Go to `web` folder create a new foloder named 'logs' and create a file named 'logs.log' Step 2- Go to `web/web` create `.env` file and fill the following information: ``` SECRET_KEY='xxxxxx' ENGINE="django.db.backends.sqlite3" NAME="db.sqlite3" USER="user" PASSWORD="password" HOST="localhost" PORT=5432 ``` You can change them if you want to use anyother databases Step 3- Run the following commands: ``` python manage.py collectstatic python manage.py makemigrations python manage.py migrate --run-syncdb python manage.py createsuperuser # if you want to ingest msmarco sample data to database python manage.py shell < fixtures/ingest_msmarco_data.py python manage.py runserver ``` Step 4- Open http://0.0.0.0:8000/
